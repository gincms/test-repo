Git in

Gin is a Git-integrated CMS. It keeps everything under version-control, meaning that you can rewind to any moment in the existence of your content — or configuration — at will. It's like being able to travel through time. When using Gin, you basically have superpowers.

Awesome.

Get started

Login

To start using Gin, you need to go to the CMS login via GithHub. If you're not signed in and don't have an account, you'll be given the opportunity to create one.

Create a project

Once logged in, you'll be able to create a new project. Projects contain all your content, as well as all your Gin CMS settings.

All changes in Gin are tracked locally within the context of your project. You can then sync your project — in other words, back-up your latest changes to the cloud — whenever you want. All you need is an internet connection.

This means that almost all work in Gin can be done offline. You only require an internet connection for three actions: login (which only has to be done once), creating projects, and syncing.

Work offline. Rewind time. This is the Gin way.

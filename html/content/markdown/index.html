Markdown allows you to format text using a very simple syntax. The Markdown formatted text is only converted to HTML once it's necessary to display it. Below is a quick guide to using Markdown to create your content. For each element, the code is presented first, followed by the resulting output.

Note: pay attention to the single space between opening symbols (e.g. *,#) and text.
